# To run locally


## Run:
1. Clone and install packages
   ```
   git clone https://github.com/derekc2205/get-a-life-ui.git
   npm install
   ```

2. Start project
   ```
   npm start
   ```


## Test:
1. Jest Test
   ```
   npm test
   ```

2. Cypress Test
   1. Start **API** test environment in `get-a-life-api`
      ```
      npm run start:e2e
      ```
   2. Start UI
      ```
      npm run
      ```
   3. Start Cypress Test
      ```
      npm run test:cypress
      ```
